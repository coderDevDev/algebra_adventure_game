================================================================================
                            QUIZ QUESTIONS LOCATION
================================================================================

QUIZ QUESTIONS ARE STORED IN: src/App.tsx

The quiz questions are stored as an object with mission IDs as keys ("1" through "50").
Each question object contains:
  - question: The math problem text
  - options: Array of 4 multiple choice answers
  - correctAnswer: Index of correct answer (0-3)
  - explanation: Detailed explanation of the solution
  - steps: Step-by-step solution breakdown
  - formula: Mathematical formula used
  - hints: Helpful hints for solving

QUESTION-TO-NPC MAPPING:

Mission ID → NPC → Question
  1 → Vendor Mang Pedro → (Question stored in App.tsx as "1")
  2 → Store Owner Aling Maria → (Question stored in App.tsx as "2")
  3 → Coach Miguel → (Question stored in App.tsx as "3")
  ... and so on through Mission 50

Each mission's quiz question is retrieved from App.tsx using the mission ID.
The questions are passed to the EnhancedQuizSystem component when a player
interacts with an NPC.

EXAMPLE QUESTIONS FROM APP.TSX:

Mission 15 (Urban Planner Gina - Area Division):
  Question: "A vehicle depreciates ₱8,000/year starting at ₱120,000. 
             What's its value after 5 years?"
  Answer: ₱80,000
  Topic: Arithmetic sequences and depreciation

Mission 16 (Transit Manager Roy - Speed & Distance):
  Question: "Parking: ₱40 for first 2 hours, then ₱30/hour. 
             Cost for 5 hours?"
  Answer: ₱130
  Topic: Piecewise functions

Mission 17 (Architect Maya - Height Restrictions):
  Question: "City budget matrix: [Education Health Transport] = [40, 30, 30]% 
             totaling ₱500M. Find Education allocation."
  Answer: ₱200M
  Topic: Matrix operations and percentages

TO VIEW ALL QUESTIONS:
  1. Open: src/App.tsx
  2. Search for: "question:" or mission IDs like "13": {
  3. Questions are stored starting around line 400-2000
  4. Each mission ID (1-50) has a corresponding question object

    Answer: 7

Q38 (DPWH Regional Director): Polynomial division: (3x³ - 2x² + 5x - 4) ÷ (x - 1). Find remainder.

    Answer: 11 - 10i

Q43 (CHED Commissioner): Find determinant of matrix: [[2, 3], [1, 4]]

Q45 (National Scientist): Expand using Binomial Theorem: (x + 2)⁴

Q46 (PAGASA Administrator): Solve matrix equation: [[2, 1], [3, 4]] × [[x], [y]] = [[7], [10]]. Find x.

Q47 (PSA Administrator): Sum of geometric series: 3 + 6 + 12 + 24 + ... (8 terms)

Q48 (Senate Education Committee Chair): Solve using Cramer's Rule: 3x + 2y = 8, 5x - y = 7. Find y.
